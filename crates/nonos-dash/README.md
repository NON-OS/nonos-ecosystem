# nonos-dash

Terminal dashboard for NONOS node operators.

## Features

- Real-time network stats
- Peer connections
- Bandwidth monitoring
- Identity management
- Log viewer
- Reward tracking

## Usage

```bash
nonos-dash
```

Or launch from daemon:
```bash
nonos-daemon dash
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Tab` | Switch panel |
| `q` | Quit |
| `r` | Refresh |
| `h` | Help |
| `1-6` | Jump to tab |

## Tabs

1. **Overview** - Node status, uptime, rewards
2. **Network** - Peers, bandwidth, circuits
3. **Identities** - ZK identity management
4. **Logs** - Live log stream
5. **Settings** - Configuration
6. **Help** - Key bindings

## Requirements

Terminal with 256-color support.
Minimum size: 80x24
